# Dependancies
- rsg-core
- ox_lib
- rNotify

# Installation
- ensure that the dependancies are added and started
- add rex-mining to your resources folder
- add rex-mining.sql to your database
- add items to your rsg-core\shared\items.lua
- add images to your rsg-inventory\html\images

# Starting the resource
- add the following to your server.cfg file : ensure rex-mining

# Admin Commands
- /createrocknode creates a rock node which is saved to the database
- /createsaltrocknode creates a salt rock node which is saved to the database
